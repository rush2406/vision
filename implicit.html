ivy_vision.create_sampled_pixel_coords_image(image_dims, samples_per_dim, batch_shape=None, normalized=False, randomize=True, homogeneous=False, dev_str='cpu')[source]¶

Create image of randomly sampled homogeneous integer \(xy\) pixel co-ordinates \(\mathbf{X}\in\mathbb{Z}^{h×w×3}\), stored as floating point values. The origin is at the top-left corner of the image, with \(+x\) rightwards, and \(+y\) downwards. The final homogeneous dimension are all ones. In subsequent use of this image, the depth of each pixel can be represented using this same homogeneous representation, by simply scaling each 3-vector by the depth value. The final dimension therefore always holds the depth value, while the former two dimensions hold depth scaled pixel co-ordinates.

[reference] deduction from top of page 154, section 6.1, equation 6.1

Parameters

  • image_dims (sequence of ints.) – Image dimensions.

  • samples_per_dim (sequence of ints.) – The number of samples to perform per image dimension.

  • batch_shape (sequence of ints, optional) – Shape of batch. Assumed no batch dimensions if None.

  • normalized (bool, optional) – Whether to normalize x-y pixel co-ordinates to the range 0-1. Default is False.

  • randomize (bool, optional) – Whether to randomize the sampled co-ordiantes within their window. Default is True.

  • homogeneous – Whether the pixel co-ordinates should be 3D homogeneous or just 2D. Default is True.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc.

Type

homogeneous: bool, optional

Returns

Image of homogeneous pixel co-ordinates [batch_shape,height,width,3]

ivy_vision.downsampled_image_dims_from_desired_num_pixels(image_dims, num_pixels, maximum=False)[source]¶

Compute the best downsampled image dimensions, given original image dimensions and the ideal total number of pixels for the downsampled image. The final number of pixels in the downsampled image dimensions may not be exact, but will best balance aspect ratio preservation and total number of pixels.

Parameters

  • image_dims (sequence of ints.) – The image dimensions of the original image.

  • num_pixels (int) – The ideal numbr of pixels in the downsampled image.

  • maximum (bool, optional) – Whether the number of pixels is a hard maximum. Default is False.

Returns

The image dimensions for a new downsampled image.

ivy_vision.stratified_sample(starts, ends, num_samples, batch_shape=None)[source]¶

Perform stratified sampling, between start and end arrays. This operation divides the range into equidistant bins, and uniformly samples value within the ranges for each of these bins.

Parameters

  • starts (array) – Start values [batch_shape]

  • ends (array) – End values [batch_shape]

  • num_samples (int) – The number of samples to generate between starts and ends

  • batch_shape (sequence of ints, optional) – Shape of batch, Inferred from inputs if None.

Returns
