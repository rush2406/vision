Implicit¶

ivy_vision.implicit.stratified_sample(starts, ends, num_samples, batch_shape=None)[source]¶

Perform stratified sampling, between start and end arrays. This operation divides the range into equidistant bins, and uniformly samples value within the ranges for each of these bins.

Parameters

  • starts (array) – Start values [batch_shape]

  • ends (array) – End values [batch_shape]

  • num_samples (int) – The number of samples to generate between starts and ends

  • batch_shape (sequence of ints, optional) – Shape of batch, Inferred from inputs if None.

Returns

The stratified samples, with each randomly placed in uniformly spaced bins [batch_shape,num_samples]

ivy_vision.implicit.render_rays_via_quadrature_rule(radial_depths, features, densities)[source]¶

Render features onto the image plane, given rays sampled at radial depths with readings of feature values and densities at these sample points.

Parameters

  • radial_depths (array) – Radial depth values [batch_shape,num_samples_per_ray+1]

  • features (array) – Feature values at the sample points [batch_shape,num_samples_per_ray,feat_dim]

  • densities (array) – Volume density values at the sample points [batch_shape,num_samples_per_ray]

Returns

The feature renderins along the rays, computed via the quadrature rule [batch_shape,feat_dim]
