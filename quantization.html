Quantization¶

Collection of Quantization Functions

ivy_vision.quantization.quantize_to_image(pixel_coords, final_image_dims, feat=None, feat_prior=None, with_db=False, pixel_coords_var=0.001, feat_var=0.001, pixel_coords_prior_var=1000000000000.0, feat_prior_var=1000000000000.0, var_threshold=(0.001, 1000000000000.0), uniform_pixel_coords=None, batch_shape=None, dev_str=None)[source]¶

Quantize pixel co-ordinates with d feature channels (for depth, rgb, normals etc.), from images \(\mathbf{X}\in\mathbb{R}^{input\_images\_shape×(2+d)}\), which may have been reprojected from a host of different cameras (leading to non-integer pixel values), to a new quantized pixel co-ordinate image with the same feature channels \(\mathbf{X}\in\mathbb{R}^{h×w×(2+d)}\), and with integer pixel co-ordinates. Duplicates during the quantization are either probabilistically fused based on variance, or the minimum depth is chosen when using depth buffer mode.

Parameters

  • pixel_coords (array) – Pixel co-ordinates [batch_shape,input_size,2]

  • final_image_dims (sequence of ints) – Image dimensions of the final image.

  • feat (array, optional) – Features (i.e. depth, rgb, encoded), default is None. [batch_shape,input_size,d]

  • feat_prior (array or float to fill with) – Prior feature image mean, default is None. [batch_shape,input_size,d]

  • with_db (bool, optional) – Whether or not to use depth buffer in rendering, default is false

  • pixel_coords_var (array or float to fill with) – Pixel coords variance [batch_shape,input_size,2]

  • feat_var (array or float to fill with) – Feature variance [batch_shape,input_size,d]

  • pixel_coords_prior_var (array or float to fill with) – Pixel coords prior variance [batch_shape,h,w,2]

  • feat_prior_var (array or float to fill with) – Features prior variance [batch_shape,h,w,3]

  • var_threshold (array or sequence of floats to fill with) – Variance threshold, for projecting valid coords and clipping [batch_shape,2+d,2]

  • uniform_pixel_coords (array, optional) – Homogeneous uniform (integer) pixel co-ordinate images, inferred from final_image_dims if None [batch_shape,h,w,3]

  • batch_shape (sequence of ints, optional) – Shape of batch. Assumed no batches if None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Quantized pixel co-ordinates image with d feature channels (for depth, rgb, normals etc.) [batch_shape,h,w,2+d], maybe the quantized variance, [batch_shape,h,w,2+d], and scatter counter image [batch_shape,h,w,1]
