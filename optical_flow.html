Optical flow¶

Collection of Optical Flow Functions

ivy_vision.depth_from_flow_and_cam_mats(flow, full_mats, inv_full_mats=None, camera_centers=None, uniform_pixel_coords=None, triangulation_method='cmp', batch_shape=None, image_dims=None, dev_str=None)[source]¶

Compute depth map \(\mathbf{X}\in\mathbb{R}^{h×w×1}\) in frame 1 using optical flow \(\mathbf{U}_{1→2}\in\mathbb{R}^{h×w×2}\) from frame 1 to 2, and the camera geometry.

Parameters

  • flow (array) – Optical flow from frame 1 to 2 [batch_shape,h,w,2]

  • full_mats (array) – Full projection matrices [batch_shape,2,3,4]

  • inv_full_mats (array, optional) – Inverse full projection matrices, inferred from full_mats if None and ‘cmp’ triangulation method [batch_shape,2,3,4]

  • camera_centers (array, optional) – Camera centers, inferred from inv_full_mats if None and ‘cmp’ triangulation method [batch_shape,2,3,1]

  • uniform_pixel_coords (array, optional) – Homogeneous uniform (integer) pixel co-ordinate images, inferred from image_dims if None [batch_shape,h,w,3]

  • triangulation_method (str, optional) – Triangulation method, one of [cmp|dlt], for closest mutual points or homogeneous dlt approach, closest_mutual_points by default

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • image_dims (sequence of ints, optional) – Image dimensions. Inferred from inputs in None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Depth map in frame 1 [batch_shape,h,w,1]
