ivy_vision.cuboid_signed_distances(cuboid_ext_mats, cuboid_dims, query_positions, batch_shape=None)[source]¶

Return the signed distances of a set of query points from the cuboid surfaces.

[reference]

Parameters

  • cuboid_ext_mats (array) – Extrinsic matrices of the cuboids [batch_shape,num_cuboids,3,4]

  • cuboid_dims (array) – Dimensions of the cuboids, in the order x, y, z [batch_shape,num_cuboids,3]

  • query_positions (array) – Points for which to query the signed distances [batch_shape,num_points,3]

  • batch_shape (sequence of ints, optional) – Shape of batch. Assumed no batches if None.

Returns

The distances of the query points from the closest cuboid surface [batch_shape,num_points,1]
