pixel_cost_volume¶

ivy_vision.pixel_cost_volume(image1, image2, search_range, batch_shape=None)[source]¶

Compute cost volume from image feature patch comparisons between first image \(\mathbf{X}_1\in\mathbb{R}^{h×w×d}\) and second image \(\mathbf{X}_2\in\mathbb{R}^{h×w×d}\), as used in FlowNet paper.

[reference]

Parameters

  • image1 (array) – Image 1 [batch_shape,h,w,D]

  • image2 (array) – Image 2 [batch_shape,h,w,D]

  • search_range (int) – Search range for patch comparisons.

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

Returns

Cost volume between the images [batch_shape,h,w,(search_range*2+1)^2]
