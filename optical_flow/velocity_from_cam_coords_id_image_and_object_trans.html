velocity_from_cam_coords_id_image_and_object_trans¶

ivy_vision.velocity_from_cam_coords_id_image_and_object_trans(cam_coords_t, id_image, obj_ids, obj_trans, delta_t, batch_shape=None, image_dims=None, dev_str=None)[source]¶

Compute velocity image from co-ordinate image, id image, and object transformations.

Parameters

  • cam_coords_t (array) – Camera-centric homogeneous co-ordinates image in frame t [batch_shape,h,w,4]

  • id_image (array) – Image containing per-pixel object ids [batch_shape,h,w,1]

  • obj_ids (array) – Object ids [batch_shape,num_obj,1]

  • obj_trans (array) – Object transformations for this frame over time [batch_shape,num_obj,3,4]

  • delta_t (array) – Time difference between frame at timestep t-1 and t [batch_shape,1]

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • image_dims (sequence of ints, optional) – Image dimensions. Inferred from inputs in None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Relative velocity image [batch_shape,h,w,3]
