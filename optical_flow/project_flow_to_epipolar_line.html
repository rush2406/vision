project_flow_to_epipolar_line¶

ivy_vision.project_flow_to_epipolar_line(flow, fund_mat, uniform_pixel_coords=None, batch_shape=None, image_dims=None, dev_str=None)[source]¶

Project optical flow \(\mathbf{U}_{1→2}\in\mathbb{R}^{h×w×2}\) to epipolar line in frame 1.

[reference]

Parameters

  • flow (array) – Optical flow from frame 1 to 2 [batch_shape,h,w,2]

  • fund_mat (array) – Fundamental matrix connecting frames 1 and 2 [batch_shape,3,3]

  • uniform_pixel_coords (array, optional) – Homogeneous uniform (integer) pixel co-ordinate images, inferred from image_dims if None [batch_shape,h,w,3]

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • image_dims (sequence of ints, optional) – Image dimensions. Inferred from inputs in None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Optical flow from frame 1 to 2, projected to frame 1 epipolar line [batch_shape,h,w,2]
