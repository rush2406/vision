render_implicit_features_and_depth¶

ivy_vision.render_implicit_features_and_depth(network_fn, rays_o, rays_d, near, far, samples_per_ray, render_depth=True, render_feats=True, render_variance=False, batch_size_per_query=8192, inter_feat_fn=None, with_grads=True, v=None)[source]¶

Render an rgb-d image, given an implicit rgb and density function conditioned on xyz data.

Parameters

  • network_fn (callable) – the implicit function.

  • rays_o (array) – The camera center [batch_shape,3]

  • rays_d (array) – The rays in world space [batch_shape,ray_batch_shape,3]

  • near (array) – The near clipping plane values [batch_shape,ray_batch_shape]

  • far (array) – The far clipping plane values [batch_shape,ray_batch_shape]

  • samples_per_ray (int) – The number of stratified samples to use along each ray

  • render_depth (bool, optional) – Whether to render the depth. Default is True.

  • render_feats (bool, optional) – Whether to render the features. Default is True.

  • render_variance (bool, optional) – Whether to also render the feature variance. Default is False.

  • batch_size_per_query (int, optional) – The maximum batch size for querying the implicit network. Default is 1024.

  • inter_feat_fn (callable, optional) – Function to extract interpolated features from world-coords [batch_shape,ray_batch_shape,3]

  • with_grads (bool, optional) – Whether to track gradients during the network forward pass. Defualt is True.

  • v (ivy Container of variables) – The container of trainable variables for the implicit model. default is to use internal variables.

Returns

The rendered feature [batch_shape,ray_batch_shape,feat] and radial depth [batch_shape,ray_batch_shape,1]
