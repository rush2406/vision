coord_image_to_trimesh¶

ivy_vision.coord_image_to_trimesh(coord_img, validity_mask=None, batch_shape=None, image_dims=None, dev_str=None)[source]¶

Create trimesh, with vertices and triangle indices, from co-ordinate image.

Parameters

  • coord_img (array) – Image of co-ordinates [batch_shape,h,w,3]

  • validity_mask (array, optional) – Boolean mask of where the coord image contains valid values [batch_shape,h,w,1]

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • image_dims (sequence of ints, optional) – Image dimensions. Inferred from inputs in None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Vertices [batch_shape,(hxw),3] amd Trimesh indices [batch_shape,n,3]
