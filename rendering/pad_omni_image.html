pad_omni_image¶

ivy_vision.pad_omni_image(image, pad_size, image_dims=None)[source]¶

Pad an omni-directional image with the correct image wrapping at the edges.

Parameters

  • image (array) – Image to perform the padding on [batch_shape,h,w,d]

  • pad_size (int) – Number of pixels to pad.

  • image_dims (sequence of ints, optional) – Image dimensions. Inferred from Inputs if None.

Returns

New padded omni-directional image [batch_shape,h+ps,w+ps,d]
