ivy_vision.quantize_pixel_coords(pixel_coords, prior, final_image_dims, mode='proj', with_db=False, pixel_coords_var=0.001, prior_var=1000000000000.0, var_threshold=(0.001, 1000000000000.0), uniform_pixel_coords=None, batch_shape=None, dev_str=None)[source]¶

Quantize pixel co-ordinates with d feature channels (for depth, rgb, normals etc.), from images \(\mathbf{X}\in\mathbb{R}^{input\_images\_shape×(2+d)}\), which may have been reprojected from a host of different cameras (leading to non-quantized values), to a new quantized pixel co-ordinate image with the same feature channels \(\mathbf{X}\in\mathbb{R}^{h×w×(3+d)}\), and with integer pixel co-ordinates. Duplicates during the quantization are mean averaged.

Parameters

  • pixel_coords (array) – Coordinates to scatter (depth, rgb, normals, etc.) [batch_shape,input_size,2+d]

  • prior (array or float to fill with) – Coords prior [batch_shape,h,w,d]

  • final_image_dims (sequence of ints) – Image dimensions of the final image.

  • mode (str, optional) – Rendering mode, be one [proj|omni] for projective or omni-directional rendering, default is proj

  • with_db (bool, optional) – Whether or not to use depth buffer in rendering, default is false

  • pixel_coords_var (array or float to fill with) – Spherical polar pixel co-ordinates diagonal covariance [batch_shape,input_size,d]

  • prior_var (array or float to fill with) – Coords prior diagonal covariance [batch_shape,h,w,d]

  • var_threshold (array or sequence of floats to fill with) – Variance threshold, for projecting valid coords and clipping [batch_shape,d,2]

  • uniform_pixel_coords (array, optional) – Homogeneous uniform (integer) pixel co-ordinate images, inferred from final_image_dims if None [batch_shape,h,w,3]

  • batch_shape (sequence of ints, optional) – Shape of batch. Assumed no batches if None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Quantized pixel co-ordinates image with d feature channels (for depth, rgb, normals etc.) [batch_shape,h,w,2+d] with other d scattered, and scatter counter image [batch_shape,h,w,1]
